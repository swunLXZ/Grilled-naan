# Cocos烤馕小游戏

这是一个使用Cocos Creator 2.1开发的小游戏，结合了一些非物质传统文化遗产元素，供广大游戏开发者提供一点参考。

## 运行方法
使用gitclone或是直接下载，下载完后直接在cocos打开就可以运行，注意有些版本可能无法兼容。
